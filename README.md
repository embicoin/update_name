# update_name
Qtライブラリを使ってC++作ったupdate_nameです。

## 使い方

### 起動方法

#### Linux
update_name.shを起動するだけ。

#### Mac OS X
update_nameを起動するだけ。

#### Windows
update_name.exeを	起動するだけ。

### Twitterでの使い方

nameを更新するには以下のようなツイートをしてください。

* @_your\_screen\_name_ update\_name _new\_name_  
* _new\_name_(@_your\_screen\_name_)
